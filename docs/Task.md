# Технічне Завдання (ТЗ): "Агро-Купол: Кисневий Рубіж" (Space Farm Survival)

## Концепт гри
**Жанр:** Ізометрична симулятор-ферма з елементами виживання.
**Сетинг:** Мертва безкиснева планета. Герой виживає у скляному куполі, вирощуючи космічні рослини, що генерують кисень.
**Геймплей:** Гібрид механік управління фермою та контролю життєво необхідного ресурсу (кисню), фокус на автоматизацію та прогресію бази.

## 1. Архітектура та Технологічний Стек
- **Мова/Фреймворк:** Java + LibGDX.
- **Рендер:** IsometricTiledMapRenderer для тайлової ізометричної мапи.
- **Збереження даних:** JSON (Google GSON або Jackson) для серіалізації стану гри (карта, інвентар, рівень кисню).

## 2. Ключові Геймплейні Механіки
**Кисень (OxygenLevel):**
- Глобальний параметр від 0 до 100%. Підтримується зрілими рослинами та "Кисневими пілонами".
- Поза зоною купола вмикається таймер кисневих балонів скафандра. Якщо він закінчується — гравець гине.

**Фермерство:**
- **Цикл росту:** Насіння -> Паросток -> Зріла рослина.
- **Ресурси:** Вода (видобувається з льоду у космосі) та Світло (енергія для фітоламп).
- **Рослини:**
  - *Швидка біомаса:* росте швидко, дає мінімум кисню.
  - *Повільний кисневий аероб:* росте довго, дає величезний приріст кисню.
  - *Кристалічний томат:* для продажу, дуже дорогий.

**Економіка та Прогресія:**
- *Ракета-термінал:* Гравець відправляє врожай на Землю і через певний час отримує гроші/технології для розширення (нові грядки, кращі скафандри, пілони).

## 3. Елементи Автоматизації
- **Дрони-помічники:** Створюються для автоматичного поливу та збору врожаю.
- **Навігація:** Рухаються по ізометричній сітці (використання алгоритму A* або Manhattan distance) між грядками та складом.

## 4. Покроковий план MVP
- **Етап 1:** Ізометрична сітка, камера та рух гравця.
- **Етап 2:** Грядки, посадка рослин та таймери їх росту.
- **Етап 3:** Киснева система, поділ на зони ("під куполом" та "космос").
- **Етап 4:** Ракетний термінал, економіка, прості дрони для збирання врожаю.
